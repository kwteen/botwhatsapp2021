### Requirements
  - Windows: Nodejs, Gitbash, Google Chrome
  - Linux: Nodejs, NPM, Google Chrome

### Install

```sh
$ git clone https://github.com/AkaShiro22/botwhatsapp2021.git
$ cd botwhatsapp2021
$ npm i
$ npm start
```

after running it you need to scan the qr.
