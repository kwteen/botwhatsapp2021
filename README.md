## Getting Started

This project require NodeJS v12.

### Install
Clone this project

```bash
> git clone https://github.com/AkaShiro22/botwhatsapp2021.git
> cd botwhatsapp2021
```

Install the dependencies:

```bash
> npm install 
```

### Usage
Run the Whatsapp bot

```bash
> npm start
```
